# movie-barnacle

Flutter App for Viewing Information About Movies

Versions needed to run this project
- Flutter 3.24.1
- Dart 3.5.1 

API Endpoint used for fetching data -
https://raw.githubusercontent.com/prust/wikipedia-movie-data/master/movies.json
